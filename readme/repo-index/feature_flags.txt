# L9 Feature Flags
# ==================
# Runtime feature flags for L9 capabilities.

## Active Feature Flags

| Flag | Purpose | Default | Location |
|------|---------|---------|----------|
| `L9_ENABLE_GOVERNANCE` | Feature flag | `false` | scripts/audit/tier1/audit_infrastructure_health.py |
| `L9_ENABLE_GOVERNANCE_ENFORCEMENT` | Feature flag | `false` | codegen/templates/python/module-template.py |
| `L9_ENABLE_GRAPH` | Feature flag | `false` | scripts/audit/L9_AUDIT_SUITE_EXPANSION/audit_infrastructure_health.py |
| `L9_ENABLE_LEGACY_CHAT` | Gate old apiserver.py POST /chat | `true` | api/server.py |
| `L9_ENABLE_LEGACY_SLACK_ROUTER` | Gate old webhookslack.py path | `false` | api/webhook_slack.py |
| `L9_ENABLE_MEMORY` | Feature flag | `false` | scripts/audit/L9_AUDIT_SUITE_EXPANSION/audit_infrastructure_health.py |
| `L9_ENABLE_STRICT_MODE` | Feature flag | `false` | codegen/templates/python/module-template.py |
| `L9_ENABLE_WS_ORCHESTRATOR` | WebSocket routes use wstaskrouter | `true` | tools/export_repo_indexes.py |
| `L9_NEW_AGENT_INIT` | Enable 7-phase bootstrap ceremony | `true` | api/server.py |
| `L9_USE_KERNELS` | Load kernels from files | `true` | api/llm.py |

## Flag Usage Pattern

```python
import os

if os.getenv('L9_NEW_AGENT_INIT', 'true').lower() == 'true':
    # Use new 7-phase bootstrap
    await bootstrap_agent(config, substrate)
else:
    # Legacy initialization
    agent = create_agent_legacy(config)
```
